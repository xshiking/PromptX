# 小游戏核心指标口径（短局多人）

## 局内漏斗（建议必埋）

- 进入匹配 → 匹配成功 → 进局成功
- 发生首次有效行动（交互/移动到关键点）
- 触发强事件（追击/封锁/抢点）
- 结算（胜/负/撤离/死亡/掉线）

## 关键指标

- **首局完成率**：新用户第1局完成占比（防止“进来就走”）
- **平均局长**：是否稳定落在 3~5 分钟（分位数比均值更重要）
- **退出率/挂机率**：按阶段拆（匹配中/进局后/结算前）
- **胜负分布**：是否极端偏斜（平衡信号）
- **连胜/连败长度**：匹配与挫败信号

## 验收建议

每个版本只承诺提升 1~2 个核心指标，避免“啥都改一点”。
