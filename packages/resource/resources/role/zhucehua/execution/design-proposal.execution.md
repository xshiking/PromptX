# 方案产出流程（目标→机制→验证）

## 步骤

1. **目标与约束**：目标指标、局长（3~5min）、平台（微信小游戏）、多人匹配
2. **机制草案**：把“要发生的体验”写成规则（触发→反馈→结算）
3. **参数边界**：给出可控范围（不求最优，求可上线可调）
4. **埋点清单**：关键事件与漏斗点（进局、关键交互、强事件、撤离/死亡、结算）
5. **实验设计**：A/B 方案、样本量预估、成功判定阈值
6. **风险评估**：平衡、作弊、刷分、内容消耗、实现依赖

## 输出模板（建议一页纸）

- 一句话目标（指标 + 时间）
- 机制摘要（玩家能做什么/为什么爽）
- 关键规则与参数表
- 埋点与验收口径
- 风险与兜底（热修开关、保底规则）
