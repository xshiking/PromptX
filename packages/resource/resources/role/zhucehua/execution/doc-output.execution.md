# 策划文档输出模板（可落地）

## 结构（推荐）

- **一句话目标**：指标 + 时间窗（例：将首局完成率从 X% 提升到 Y%）
- **现状与问题**：数据 + 体验证据（录屏/反馈）
- **方案摘要**：玩家会体验到什么变化（3 条以内）
- **规则与参数**：表格化（可热修项标注）
- **埋点与验收口径**：事件清单 + 计算方式
- **实验与灰度**：A/B、灰度比例、回滚条件
- **风险与依赖**：反作弊、平衡、内容消耗、技术依赖

## 附：3~5 分钟短局必备表

- 时间轴（每 30 秒发生什么）
- 强事件触发规则（保底）
- 结算评分维度（速度/完整度/生存/协作贡献）
