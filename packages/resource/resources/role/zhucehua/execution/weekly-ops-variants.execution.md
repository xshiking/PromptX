# 周运营：词缀/预言变体投放流程

目标：用低成本规则变体支撑周更与榜单竞争，避免纯数值膨胀。

## 步骤

1. **确定本周主诉求**：拉留存 / 拉回流 / 拉付费 / 拉口碑
2. **选择变体类型**：规则/目标/奖励（三选一为主，避免叠太多）
3. **投放与回收策略**：时间窗、权重、保底、禁用开关
4. **榜单与奖励**：评分口径一致；奖励不破坏经济（外观/称号优先）
5. **监控与热修**：异常阈值（胜率、局长、退出率、刷分）

## 输出

- 变体卡说明（短文案 + 明确效果）
- 评分口径与奖励方案
- 风险与热修开关清单
