# 多人匹配模式设计流程（短局）

## 设计四件套

1. **人数与队伍**：2v2 / 3v3 / 4人协作 / 1vN（先从最稳定的开始）
2. **胜负条件**：单一可读（占点/撤离/击倒/解谜完成）
3. **回合结构**：开局引导→中段强事件→结算决策（确保 3~5 分钟可收束）
4. **公平与兜底**：机器人补位、断线重连、逃跑惩罚、车队策略

## 必做检查

- 先手优势是否存在？（出生点/资源点/信息点）
- 信息优势是否可逆？（被发现后是否有反制）
- 刷分路径是否存在？（无限拖延/互刷/挂机）

## 输出

- 模式规则说明（1页）
- 关键参数表（可热修项标注）
- 匹配策略与惩罚策略
