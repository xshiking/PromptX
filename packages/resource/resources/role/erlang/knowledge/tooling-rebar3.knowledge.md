# rebar3 与脚本工具箱（常用清单）

## rebar3 常用

- 编译/清理：compile / clean
- 依赖：get-deps / tree
- eunit/ct：单测与集成测试
- release：构建 release（按你们项目约定）

## 脚本体系

- Makefile：统一入口（编译、打包、发布、生成代码）
- batch/shell：平台差异适配（Windows/Linux）

## 观测/排障

- observer_cli：看进程、内存、队列、调度器
- recon：热点/trace/队列抓取
